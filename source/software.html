Программное обеспечение

Reg.place — это сервис для регистрации и управления спортивными мероприятиями. Он предлагает функции для соревнований, клубов, тренеров, волонтеров и включает варианты подписок, абонементов и подарочных сертификатов. На сайте есть поиск мероприятий по времени, виду спорта и местоположению, что упрощает регистрацию на различные мероприятия.

Программное обеспечение

  • использование веб-технологий для создания интерфейса пользователя
  • системы управления базами данных для хранения данных о событиях и участниках
  • интеграцию с платежными системами для обработки финансовых транзакций

Возможности ИТ-продукта

Сервис состоит из нескольких блоков:

1. Предмет договора

Публичная часть, доступная всем пользователям

– незарегистрированные пользователи могут просматривать информацию об опубликованных соревнованиях и искать нужные по своим параметрам

– для регистрации на мероприятие пользователь должен иметь учетную запись в системе, содержащую его основные данные. Наличие профиля позволяет упростить регистрацию на мероприятия и в большинстве случаев регистрироваться буквально в один-два клика

– при регистрации система проверяет соответствие пользователя критериям регистрации, установленным организатором

Раздел для организаторов, где пользователи-организаторы создают и настраивают мероприятия для регистрации

– настройка сроков и условий регистрации, стоимости взносов, политика изменения цены в зависимости от различных факторов, политика скидок и промо-акций

– настройка анкеты для регистрации, необходимых лимитов, полей, документов для загрузки и т.п.

– статистика регистраций и управление списками участников

– финансовый учет регистраций и управление собранными средствами

Административный раздел

– позволяет администраторам системы помогать организаторам в настройке мероприятий, подключать дополнительные опции или функции для регистраций

Языки программирования: Ruby, Javascript, SQL Стэк: Rails, Vue, PostgreSQL, Redis

Преимущества ИТ-решения

  • Управление базой участников: детальная статистика и автоматизация жизненного цикла мероприятия
  • Сервис подписок на регулярные мероприятия
  • Промокоды, реферальные ссылки
  • Поддержка мероприятия: ответы на вопросы участников об оплате, возвратах, технических деталях, возможность организации горячей линии для организатора
  • Возможности API для интеграции с внешними сервисами (как в качестве клиента, так и в качестве сервера)
